# Dataset-Narkotika_355_369

### Scraping Putusan Pidana Khusus
Scraping Data Pengadilan Negeri Jombang Mengenai Pidana Khusus tentang Narkotika dan Psikotropika tahun 2023.

### Dataset
Dataset yang kami gunakan untuk scraping ini adalah 'Pengadilan Negeri Jombang' mengenai 'Pidana Khusus' tentang 'Narkotika dan Psikotropika' tahun 2023. Dimana hasil scraping tersebut berjumlah 168 data.
